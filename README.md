# E-Commerce Sales & Customer Insights (SQL Project)

## Overview
This project analyzes e-commerce sales data using PostgreSQL to uncover
revenue trends, customer behavior, product performance, and delivery efficiency.

## Tools Used
- PostgreSQL
- pgAdmin 4
- SQL

## Dataset
Brazilian E-Commerce Public Dataset by Olist from Kaggle
Includes customers, orders, products, sellers, and payments data.

## Key Analysis
- Total and monthly revenue analysis
- Customer repeat behavior
- Best-selling product categories
- Delivery performance metrics
- Advanced SQL using CTEs and window functions

## SQL Concepts Used
- JOINs (INNER)
- GROUP BY & HAVING
- Aggregate Functions
- CASE WHEN
- CTEs
- Window Functions (RANK, RUNNING TOTAL)

## How to Run
1. Create database in PostgreSQL
2. Create tables using schema
3. Import CSV files
4. Run queries from `analysis.sql`

## Key Insights
- Identified top revenue-generating categories
- Analyzed customer retention patterns
- Evaluated logistics performance
